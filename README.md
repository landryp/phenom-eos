# phenom-eos

Code for reproducing phenomenological models of candidate EoSs, like piecewise polytropes.

---

### Scripts

###### Piecewise polytrope (pwp)

* geteos_pwp

###### Spectral EoS (spec)

* geteos_spec

###### Strange quark matter EoS (sqm)

* geteos_sqm

###### Constant sound speed hadron-quark hybrid EoS (css)

* geteos_css

---

### Tools

###### EoS plot

* ploteos eos.csv

###### Affix SLy crust

* addcrust eos.csv
